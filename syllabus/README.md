# Syllabus — Chapter Outlines and Source References

This folder contains the teaching outline (教案大纲) for each chapter. Each file lists:

- **Learning objectives** — what the reader should take away
- **Section outline** — the structural skeleton of the chapter
- **Primary sources** — which textbook chapters and papers the outline draws from
- **Status** — whether the chapter is complete or planned

These outlines serve as the blueprint for chapter drafting and review. They are not rendered into the site.
